Szybkie uruchomienie programu:
1. npm_install.bat (tylko dla pierwszego uruchomienia)
2. start.bat

Appsettings.json:
-Do działania niektórych funkcji aplikacji (np. wysyłania email) potrzebny jest własny connectionString z azure.
-Do poprawnego działania aplikacji potrzebny jest postgres ze stworzoną bazą danych zgodną z appsettings, z aktualną migracją.
